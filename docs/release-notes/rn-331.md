### Magic Button on Original Email

The magic button is now available when viewing the original email from a briefing card. Previously, you had to navigate back to take action — now you can reply, forward, or act directly from the email view.

### Gmail Category Filtering

Clairify no longer syncs emails from Gmail's Promotions, Social, Updates, and Forums categories into your Instack. These emails are now filtered out automatically, honoring the same categories Gmail already separates for you.

### Email Blocking Rules

You can now block unwanted emails from syncing into Clairify. Block by Gmail category, sender address, or subject keyword — configurable from the web admin panel.

### PDF Attachments (Gmail)

You can now access PDF attachments directly from Clairify. For emails with attachments, the PDF is viewable without leaving the app.

### Prompt Writing Guidance

The web admin prompt editor now includes inline guidance for writing valid summarization prompts. If a prompt is rejected, the error toast links directly to the [prompt guidelines documentation](../writing-prompts.md). Tooltips on the prompt form also link to the documentation.
